Sobre

Este site foi desenvolvido como Trabalho Prático 1 da disciplina de Desenvolvimento de Interfaces Web do 1° período do curso de Engenharia de Software da PUC Minas cujo professor orientador é Diego Augusto de Faria Barros. O objetivo é apresentar um guia gastronômico de Belo Horizonte, destacando restaurantes de diferentes categorias e estilos culinários.

O projeto utiliza HTML, CSS, JavaScript e Bootstrap contendo responsividade, além de dados armazenados em estrutura JSON e processados dinamicamente através do JavaScript.
